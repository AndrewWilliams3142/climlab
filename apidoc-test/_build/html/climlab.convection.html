Convective Adjustment to a specified lapse rate.

Input argument lapserate gives the lapse rate expressed in degrees K per km (positive means temperature increasing downward).

Default lapse rate is 6.5 K / km.

Returns the adjusted Column temperature. inputs: p is pressure in hPa T is temperature in K c is heat capacity in in J / m**2 / K
